• git init - створює репозиторій у місці виклику
  • git add - додає зміни з робочого каталогу до проіндексованих файлів Git
  • git config - використовується для налаштування Git
  • git rm - видаляє файли з Git-репозиторію
  • git status - показує статус файлів у робочому каталозі
  • git commit - створює новий коміт, який зберігає зміни в репозиторії Git
  • git brunch - створює нову гілку в репозиторії Git
  • git checkout - перемикає робочу директорію на вказану гілку
  • git push - надсилає локальні зміни в віддалений репозиторій
  • touch - створює новий файл
  • reset - очищає консоль терміналу
